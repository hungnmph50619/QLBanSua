# QLBanSua

Phần mềm quản lý cửa hàng sữa được xây dựng bằng C# WinForms.

## Công nghệ sử dụng
- C#
- WinForms
- SQL Server
- ADO.NET

## Chức năng chính
- Quản lý hàng hóa
- Quản lý hóa đơn
- Quản lý khách hàng
- Quản lý nhân viên
- Thống kê doanh thu

## Cấu trúc project
- GUI_QLBanSua: giao diện người dùng
- DAL_QLBanHang: truy cập dữ liệu
- BUS_QLBanHang: xử lý nghiệp vụ
- DTO_QLBanHang: đối tượng dữ liệu

## Cách chạy project
1. Mở file `QLBanSua.sln` bằng Visual Studio
2. Cấu hình lại chuỗi kết nối cơ sở dữ liệu
3. Khởi chạy project
